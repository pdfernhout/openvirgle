Variations on theme of Pointrel triples (RDF-like) from a CGI Semantic Wiki perspective.
The version of Pointrel used is from here (and is designed more for CGI and transactions):
  http://sourceforge.net/project/showfiles.php?group_id=6010
  Pointrel20030812.2 For Py

This code as-is is not safe to run on a world exposed web server because it allows creating and running arbitrary Python programs on the server.
It is a tiny bit like Zope in some ways, and could move a little in that direction to become more secure. :-)
